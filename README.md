# 网页端哔哩哔哩净流

一个用于网页端哔哩哔哩的 Tampermonkey / 篡改猴用户脚本。

作者：千林  
联系方式：QQ 1817716374

## 脚本作用

- 过滤哔哩哔哩首页推荐流中的广告卡片。
- 过滤首页带“小火箭”等推广标识的推流内容。
- 自动补齐首页推荐流：拦截原始推荐接口，额外获取一些内容，过滤广告后返回正常视频。
- 隐藏并移除视频播放页右侧广告。
- 隐藏并移除游戏视频页常见的游戏下载、评分、公告等推广卡片。
- 隐藏并移除视频下方的横幅广告、游戏活动推广横幅。

## 安装方式

### 方式一：通过 Raw 链接安装

1. 浏览器安装并启用 Tampermonkey / 篡改猴。
2. 打开安装链接：<https://github.com/1817716374/网页端哔哩哔哩净流/raw/main/bilibili-clean-feed.user.js>
3. 篡改猴会自动弹出安装页面。
4. 点击安装，然后刷新哔哩哔哩页面。

如果浏览器没有自动弹出安装页，可以打开仓库中的 `bilibili-clean-feed.user.js` 文件，再点击 GitHub 页面右上方的 `Raw`。

### 方式二：手动新建脚本

1. 打开篡改猴管理面板。
2. 点击“添加新脚本”。
3. 删除默认内容。
4. 将 `bilibili-clean-feed.user.js` 的完整内容粘贴进去。
5. 保存脚本。
6. 刷新哔哩哔哩页面。

## 使用范围

脚本仅匹配：

```text
https://www.bilibili.com/*
```

当前主要处理：

- 哔哩哔哩首页。
- 哔哩哔哩视频播放页。

## 注意事项

- 如果更新脚本后页面仍显示旧效果，请按 `Ctrl + F5` 强制刷新页面。
- 首页推荐内容会保留用户登录态和 Cookie，以尽量维持个性化推荐准确性。
- 哔哩哔哩页面结构可能会变化，如果失效，需要根据新的页面结构继续维护。
- 本脚本只处理网页端，不处理手机 App。
